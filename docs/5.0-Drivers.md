# Drivers

NovariaOS drivers run in kernel space and are registered as block or character devices accessible through the VFS.

## Structure

| Section | Description              |
|---------|--------------------------|
| 5.1     | IDE/ATA disk driver      |
| 5.2     | Block device abstraction |
| 5.3     | Ramdisk                  |
| 5.4     | Keyboard                 |
| 5.5     | Serial (COM1)            |
| 5.6     | NVMe driver              |
| 5.7     | PCI subsystem            |

## Block devices

Block devices are registered via `register_block_device` and accessed through the block device abstraction layer. FAT32 and other filesystems mount block devices by name (e.g. `hda`).

## Character devices

Character devices are registered in devfs under `/dev/`. See [4.3-devfs](4.3-devfs.md).
