# Build from source
To build avariaOZV, first you need to grab the sources from the avariaOZV Github repository:
```shell
[user@pc: ~] $ git clone https://github.com/avariaOZV/avariaOZV-src.git
```

You will then need to install all dependencies:
- gcc
- nasm
- libc6-dev
- limine
- gcc-multilib
- build-essential
- xorriso
- qemu-system-x86_64
- [chorus](https://github.com/z3nnix/chorus/)

When you have installed all the necessary dependencies, it's time to build the project.

To build only:
```shell
[user@pc: ~/avariaOZV] $ chorus all
```

To build and run:
```shell
[user@pc: ~/avariaOZV] $ chorus all run
```
