# Iso9660

ISO 9660 is a read-only filesystem standard for CD-ROM images. NovariaOS uses it as the boot medium — the kernel and initial programs are loaded from an ISO image.

## Key features

- Read-only access to files and directories
- Block size: 2048 bytes (configurable from PVD)
- Filenames are normalized to lowercase
- Version suffixes (e.g. `;1`) are stripped automatically
- Mounted into VFS at boot time

## Structure

### Primary Volume Descriptor (PVD)

Located at LBA 16. Identifies the filesystem and provides a pointer to the root directory entry.

| Field                   | Description                        |
|-------------------------|------------------------------------|
| `type`                  | `1` for primary volume descriptor  |
| `identifier`            | Always `"CD001"`                   |
| `logical_block_size_le` | Block size in bytes (usually 2048) |
| `root_directory_entry`  | 34-byte inline root dir entry      |

### Directory Entry

Each file or directory is described by a directory entry:

| Field       | Description                         |
|-------------|-------------------------------------|
| `extent_le` | LBA of file/directory data          |
| `size_le`   | Data size in bytes                  |
| `flags`     | `0x02` = directory, `0x01` = hidden |
| `name_len`  | Length of file identifier           |

## VFS integration

At boot, `iso9660_mount_to_vfs` recursively walks the ISO directory tree and registers all files and directories into the VFS. Files larger than `MAX_FILE_SIZE` (65536 bytes) are skipped.

## API

| Function                 | Description                                  |
|--------------------------|----------------------------------------------|
| `iso9660_init`           | Initialize from raw ISO data in memory       |
| `iso9660_find_file`      | Find a file by path, return pointer and size |
| `iso9660_list_dir`       | Print directory contents to kernel log       |
| `iso9660_mount_to_vfs`   | Mount ISO subtree into VFS at a given path   |
| `iso9660_is_initialized` | Check if the filesystem has been initialized |
