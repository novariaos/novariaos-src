# Keyboard Driver

PS/2 keyboard driver with scancode translation, modifier tracking, and a circular input buffer.

## Ports

| Port   | Description          |
|--------|----------------------|
| `0x60` | Data port (scancodes) |
| `0x64` | Status port           |

## Input buffer

Circular buffer of `KEYBOARD_BUFFER_SIZE` (256) characters. Overflow is silently discarded.

| Function              | Description                                       |
|-----------------------|---------------------------------------------------|
| `keyboard_init`       | Initialize buffer and state                       |
| `keyboard_has_char`   | Poll the port and return true if data is available |
| `keyboard_getchar`    | Block until a character is available, return it   |
| `keyboard_getline`    | Read a line, echo to screen, stop on `\n`         |

`keyboard_getchar` calls `nvm_scheduler_tick` while waiting, so other NVM processes continue to run.

## Scancode translation

Set 1 scancodes (XT). Two tables are used:

- `scancode_to_ascii` — normal
- `scancode_to_ascii_shifted` — with Shift held

### Modifier keys

| Key        | Scancode make | Effect                            |
|------------|---------------|-----------------------------------|
| Left Shift | `0x2A`        | `shift_pressed = true`            |
| Right Shift| `0x36`        | `shift_pressed = true`            |
| Ctrl       | `0x1D`        | `ctrl_pressed = true`             |
| Alt        | `0x38`        | `alt_pressed = true`              |
| Caps Lock  | `0x3A`        | Toggle `caps_lock`                |

Break codes (bit 7 set) clear the corresponding modifier.

### Ctrl+key

When Ctrl is held, letter keys produce control characters (`'a'` → `0x01`, `'b'` → `0x02`, ...).

## Hotkeys

Up to 16 hotkeys can be registered. A hotkey fires a callback when a specific scancode + modifier combination is pressed.

```c
int  keyboard_register_hotkey(int scancode, int modifiers,
                               void (*callback)(void*), void* data);
void keyboard_unregister_hotkey(int id);
```

Modifier flags:

| Constant           | Value |
|--------------------|-------|
| `HOTKEY_MOD_SHIFT` | `1`   |
| `HOTKEY_MOD_CTRL`  | `2`   |
| `HOTKEY_MOD_ALT`   | `4`   |
