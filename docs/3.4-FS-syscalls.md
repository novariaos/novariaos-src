# Filesystem syscalls

## OPEN

Opens a file and returns a file descriptor.

| Field    | Value         |
|----------|---------------|
| Number   | `0x02`        |
| Requires | `CAP_FS_READ` |

### Stack input

Path string pushed as individual bytes, terminated by `0`:

```
| 0      |  null terminator
| char N |
| ...    |
| char 0 |  first character
```

### Return value

Pushes file descriptor (`fd`) onto the stack, or `-1` on error.

---

## READ

Reads one byte from an open file descriptor.

| Field    | Value         |
|----------|---------------|
| Number   | `0x03`        |
| Requires | `CAP_FS_READ` |

### Stack input

| Position  | Description     |
|-----------|-----------------|
| `sp - 1`  | file descriptor |

### Return value

Pushes the byte value read (0–255), `0` on EOF, or `-1` on error.

---

## WRITE

Writes one byte to an open file descriptor.

| Field    | Value          |
|----------|----------------|
| Number   | `0x04`         |
| Requires | `CAP_FS_WRITE` |

### Stack input

| Position  | Description     |
|-----------|-----------------|
| `sp - 2`  | file descriptor |
| `sp - 1`  | byte value      |

### Return value

Pushes number of bytes written (`1`), or `-1` on error.

### Notes

- Writing to fd `1` (stdout) or `2` (stderr) prints the byte to the screen via `kprint`
